⚙️ // КОРИСНА МОДЕЛЬ — ЩО ЦЕ?

Корисна модель — це захист для технічних рішень: пристроїв, конструкцій, механізмів. Її ще називають «маленьким патентом».

Головні ознаки: новизна (ще ніхто такого не робив) + придатність для промислового використання.

На відміну від патенту — НЕ потрібна «винахідницький рівень» (тобто не треба бути генієм). Швидша реєстрація: ~3–6 місяців. Термін охорони: 10 років.

⚙️ // ЗАСТОСУВАННЯ

Реєстрація корисної моделі в Україні: подається до УКРПАТЕНТ. Проводиться формальна експертиза (чи правильно оформлено) — без перевірки новизни «по суті». Тому відповідальність за новизну лежить на заявнику.

Після реєстрації — свідоцтво. Власник може: використовувати, ліцензувати, продати, заборонити іншим.

🎨 // ПРОМИСЛОВИЙ ЗРАЗОК — ЩО ЦЕ?

Промисловий зразок — це захист зовнішнього вигляду виробу: форми, кольору, орнаменту, текстури. Це «патент на дизайн».

Захищає: як річ виглядає, а не як вона працює.

Термін охорони: 25 років (5 + 5-річні продовження). Реєстрація: УКРПАТЕНТ або Гаазька система ВОІВ (міжнародно).

🎨 // КРИТЕРІЇ ТА РЕЄСТРАЦІЯ

Три умови для реєстрації промислового зразка: 1) Новизна — такого зовнішнього вигляду ще не існувало. 2) Оригінальність — відрізняється від існуючих. 3) Промислова застосовність — можна виготовити серійно.

Не захищається: функціональні ознаки (тільки для роботи), нейтральні форми, що диктуються технічною функцією.
